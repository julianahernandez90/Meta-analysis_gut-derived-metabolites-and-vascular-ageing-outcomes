# Meta-analysis_gut-derived-metabolites-and-vascular-ageing-outcomes
In this research project, we systematically summarised associations between circulating gut-related metabolites and vascular ageing outcomes.
To answer the research question, Embase, MEDLINE, Cochrane CENTRAL, Web of Science, and Google Scholar were searched up to December 2025. We included observational designs and clinical trials in adults assessing associations between circulating metabolites and blood pressure (BP), hypertension, arterial stiffness, carotid-intima media thickness, and atherosclerosis-related outcomes such as coronary artery disease (CAD). Effect estimates were harmonised to reflect associations per 1 SD increase in metabolite concentration and pooled using random effects models. Risk of bias was assessed using ROB 2, ROBINS I, ROBINS E, and NOS tools, and the certainty of evidence with GRADE.
In this repository, you will find the code for the meta-analyses, leave-one-out sensitivity analysis, and publication bias assessment (both graphically and statistically). I've also provided the scripts for the harvest plot used in the qualitative synthesis.
